# AI-video-automation
Pipeline automatizado desarrollado en Python para generación de videos cortos utilizando herramientas de inteligencia artificial.

El sistema automatiza:
Generación de imágenes mediante APIs de IA,
Síntesis de voz,
Composición automática de video.

Tecnologías utilizadas:
Python,
MoviePy,
Edge TTS,
APIs de IA generativa,
Automatización de procesos.

Funcionalidades:
Procesamiento automático de escenas,
Sincronización de audio e imágenes,
Animaciones dinámicas,
Renderizado automático de videos verticales.

Posibles mejoras:
Integración con APIs de video,
Subtítulos automáticos,
Procesamiento batch,
Publicación automática.
